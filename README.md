# vinvino02/glpn-nyu Cog model

This is an implementation of [vinvino02/glpn-nyu](https://huggingface.co/vinvino02/glpn-nyu) as a Cog model. [Cog packages machine learning models as standard containers.](https://github.com/replicate/cog)

Run a prediction:

     cog predict -i image=@street.jpg
